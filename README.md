24Servos
========

24 servos controller with Microchip PIC 18F452

- main.asm	    Assembler program (MPLAB)
- init.inc	    18F452 internal register init
- init2.inc	    Sort array init
- macro.inc	    Macro
- LICENCE.txt	License
- main.HEX	    Assembled program (config bit: only PLL active)
